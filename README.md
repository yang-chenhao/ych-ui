# Yui-react

## 一套好用的 PC 端 React 组件库

![](https://img.shields.io/badge/license-MIT-000000.svg)

## 安装

```
$ npm install yui-react
$ yarn add yui-react
```

## 使用

```javascript
import * as React from 'react'
import * as ReactDOM from 'react-dom'
import { Button } from 'yui-react'

ReactDOM.render(
  <div>
    <Button>Default</Button>
  </div>,
  mountNode
)
```
